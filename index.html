1. Windows Defender Firewall

Term: A host-based firewall built into Windows.

Click to reveal definition and example.

Definition

Controls inbound and outbound network traffic on a Windows device using security rules.

Real-world example: Like a security guard at a building entrance who checks who is allowed in or out.

2. BitLocker

Term: Full-disk encryption for Windows.

Click to reveal definition and example.

Definition

Encrypts the hard drive so data cannot be easily read if the device is lost, stolen, or removed.

Real-world example: Like locking sensitive documents inside a safe instead of leaving them on a desk.

3. Microsoft Defender Antivirus

Term: Built-in malware protection.

Click to reveal definition and example.

Definition

Detects, blocks, and removes malware such as viruses, trojans, ransomware, and spyware.

Real-world example: Like a health inspector checking food for contamination before it harms people.

4. User Account Control

Term: Permission prompt for administrative actions.

Click to reveal definition and example.

Definition

Helps prevent unauthorized system changes by requiring approval before elevated actions occur.

Real-world example: Like needing a manager’s approval before opening the cash register or changing store policy.

5. Least Privilege

Term: Give users only the access they need.

Click to reveal definition and example.

Definition

A security principle where users, apps, and services receive the minimum permissions required to perform their job.

Real-world example: Like giving a student access to one classroom, not the master key to the entire campus.

6. Windows Update

Term: Patch management for Windows.

Click to reveal definition and example.

Definition

Delivers security patches, bug fixes, driver updates, and feature improvements to reduce known vulnerabilities.

Real-world example: Like repairing a broken lock after learning thieves are exploiting that exact weakness.

7. Secure Boot

Term: Startup protection against untrusted boot code.

Click to reveal definition and example.

Definition

Verifies trusted software loads during startup, helping block bootkits and rootkits before Windows fully starts.

Real-world example: Like checking an employee badge before allowing someone into a secure building before opening hours.

Teaching Tip

Layered defense matters because no single security control is perfect. Windows security works best when multiple protections overlap: firewall, antivirus, encryption, patching, least privilege, secure startup, and user approval controls.

Real-world analogy: A college campus uses fences, locks, ID badges, cameras, security guards, policies, and emergency procedures. Each layer helps reduce risk if another layer fails.
